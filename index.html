Watt Ticker WordPress Plugin.

Watt Ticker WordPress plugin calculate and displays energy saved in watts.

Shortcode

Shortcode is supported and can be used to display the watt ticker. Shortcode also integrates the ticker with backend database and also automatically updates the value of ticker fetched from database.

[ticker]

You can also use your own markup to fetch from database. You can also use your own styling even using with shortcode. Shortcode uses theme's styles by default.
